# 👻 Dumbu: Ghost Delivery

A spooky endless-runner browser game. You are Dumbu, the night-shift courier of the spirit world — deliver souls before sunrise, dodge what lurks on the road, and don't let the king catch you.

**🎮 Play it now:** https://IamDhruvDarji.github.io/dumbu-ghost-delivery/

*(Works on desktop and mobile — no install, no downloads, just tap Play.)*

## How to Play

| Action | Desktop | Mobile |
|--------|---------|--------|
| Jump / double jump | Click or ↑ | Tap |
| Slide | ↓ | Swipe down |

Deliver souls at every gate to bank points, build combos, collect ★ coins, and survive the boss encounters. Three difficulty levels — Nightmare pays double.

## Features

- Pure HTML5 canvas — the entire game is a single file, no engine, no dependencies
- Procedurally generated music and sound effects (Web Audio API — no audio files)
- Unlockable character skins with gameplay effects, purchasable with coins earned in-game
- Rival system — enter your city and race its resident spirit
- Multiple environments and difficulty modes
- Full touch controls for mobile plus keyboard support for desktop

## Tech

- Vanilla JavaScript, single-file architecture (~1,000 lines)
- HTML5 Canvas rendering with particle effects and screen shake
- Web Audio API for all synthesised sound and music
- No frameworks, no build step — open `index.html` and play
